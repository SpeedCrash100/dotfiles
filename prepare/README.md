# Prepare section
These scripts prepare system to run main scripts of the project: install git, 
prepare package managers(install AUR helper in case of Arch) and much more

## Requirements

### Outputs
After the following files must be created:


| Name          | Description                                                  |
|---------------|--------------------------------------------------------------|
| /tmp/PM       | A package manager to use to install packages                 |
| /tmp/PYTHON   | A python 3 executable to use                                 |
